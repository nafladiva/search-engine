https://news.detik.com/berita/d-5229922/5-fakta-kebakaran-di-simprug-hanguskan-motor-dan-permukiman 5 Fakta Kebakaran di Simprug Hanguskan Motor dan Permukiman Si jago merah mengamuk di Jalan Simprug Golf 2, Kelurahan Grogol Selatan, Kecamatan Kebayoran, Jakarta Selatan Puluhan rumah kontrakan dan belasan sepeda motor dilalap api. Kebakaran itu terjadi pada Senin 26 Oktober 2020 pukul 07.50 WIB Sedikitnya 20 unit damkar diturunkan ke lokasi kebakaran. "Benar kebakaran terjadi di Jalan Simprug Golf RT 10 RW 08 Kelurahan Grogol Selatan, Kecamatan Kebayoran Saat ini unit yang diturunkan ada 20 unit," ujar operator pemadam pemadam Jakarta Selatan Suparno, saat dihubungi pada Senin (26/10/2020). Berikut 5 fakta kebakaran di Simprug hanguskan motor dan permukiman: 20 Unit Damkar Dikerahkan Sebanyak 20 unit damkar diturunkan ke lokasi kebakaran di Jalan Simprug Golf, Kelurahan Grogol Selatan, Kecamatan Kebayoran, Jakarta Selatan Operator pemadam pemadam Jakarta Selatan Suparno mengatakan kondisi api masih menyala. Informasi kebakaran di Jalan Simprug Golf diterima pada pukul 07.50 WIB "Kondisi masih merah, informasi diterima pukul 07.50 WIB," tuturnya. Suparno mengatakan saat ini kondisi api masih menyala Petugas saat ini masih berupaya memadamkan api. Objek yang terbakar merupakan rumah tinggal di kawasan padat penduduk di Jalan Simprug Golf. "Rumah tinggal padat hunian," sebut Suparno Belum diketahui penyebab hingga total kerugian dari kebakaran di Jalan Simprug Golf ini. Api Padam, Asap Masih Mengepul Api berhasil dipadamkan sekitar pukul 09.15 WIB Sejumlah petugas pemadam kebakaran masih berjibaku memadamkan sisa-sisa kebakaran. Petugas juga tampak memadamkan titik-titik api untuk memastikan api tidak lagi membesar. Kepulan asap masih terlihat di lokasi Warga juga tampak berkerumun di sekitar lokasi kebakaran. Dalam kesempatan terpisah, sekuriti bernama Ahmad Rudin menceritakan api hampir menjalar warung yang berada di seberang lokasi kebakaran. "Tiba-tiba sudah ada asap pukul 07.30-an WIB Api mulai besar, mulai menjalar ke belakang Api juga sempat mau menjalar ke warung yang ada di depannya Tidak ada korban jiwa Motor, perabotan, rumah banyak yang hangus terbakar," kata Rudin di lokasi.
