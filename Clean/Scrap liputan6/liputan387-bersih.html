https://www.liputan6.com/news/read/4391509/puluhan-personel-damkar-dikerahkan-kebakaran-di-pasaraya-manggarai-padam Puluhan Personel Damkar Dikerahkan, Kebakaran di Pasaraya Manggarai Padam - News Liputan6.com Kebakaran yang terjadi di basement Pasaraya Manggarai, Jalan Sultan Agung, Pasar Manggis, Tebet, Jakarta Selatan, sudah padam. Perwira piket Pemadam Kebakaran Jakarta Selatan, Sartono mengatakan, api berhasil dipadamkan sekitar pukul 13.25 WIB Total 14 unit mobil pemadam dikerahkan menangani kebakaran tersebut. Baca Juga "Menjelang salat Zuhur, api sudah tidak muncul, kemudian melakukan pendinginan Selesai pendinginan jam 13.25 WIB," kata dian di Pasaraya Manggarai, Minggu (25/10/2020). Sartono menjelaskan, delapan panel genset mengalami korsleting sehingga memunculkan percikan api Dia mengaku sempat mengalami kendala, saat 56 personel memadamkan api. "Kendala asap menebal sangat pekat dan kami pun susah menembus Tapi kami punya solusinya, menggunakan blower portabel sebanyak lima, dan pakai smoke remover satu unit," ucap dia.
